# Lead Capture Landing Page

A modern landing page built with Next.js for lead capture and management.

## Prerequisites

- Node.js (v18 or later)
- Yarn (recommended package manager)

## Getting Started

1. Install dependencies:
```bash
yarn install
```

2. Environment Variables:
Please document any environment variables used in the project in this README. Here are the currently used environment variables:

| Variable Name               | Description |
|--------------               |-------------|
| `NEXT_PUBLIC_API_URL`       | Base URL for API endpoints   |


3. Run the development server:
```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `yarn dev` - Start the development server
- `yarn build` - Build the production application
- `yarn start` - Start the production server
- `yarn lint` - Run ESLint
- `yarn test` - Run tests
